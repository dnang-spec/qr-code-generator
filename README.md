# 🎯 QR 코드 생성기

웹사이트 링크를 QR 코드로 변환하는 간단한 웹 애플리케이션입니다.

## ✨ 특징

- 🆓 완전 무료, 무제한 생성
- 📱 모바일 친화적
- ⚡ 빠른 생성 속도
- 💾 고화질 PNG 다운로드
- 🎨 깔끔한 UI/UX

## 🚀 사용 방법

1. 웹사이트 링크를 입력하세요
2. "생성" 버튼을 클릭하거나 Enter 키를 누르세요
3. 생성된 QR 코드를 다운로드하세요

## 🌐 데모

[GitHub Pages에서 보기](https://dnang-spec.github.io/qr-code-generator/)

## 💻 로컬 실행

1. 저장소 클론
```bash
git clone https://github.com/dnang-spec/qr-code-generator.git
```

2. index.html 파일을 브라우저로 열기

## 🛠️ 기술 스택

- React 18
- Tailwind CSS
- QR Server API
